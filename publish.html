Publishing to Surge

Surge allows you to publish pages on the web for testing. Download it here.

After installing surge, call "surge" in your terminal, make sure you're in the right directory

Making a CNAME File

Creating a CNAME file lets you update your site with the same url, so you don't have a new url each time you make changes

This forum is great and explained it better than surge.sh.

Use the format "echo your-sites-name.surge.sh >CNAME" You should have a CNAME file in your directory if it works.

Use "surge ." to update it subsequentlly.
