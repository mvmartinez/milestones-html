Want to search the web? Try Google!

The tags used to make the above snippet are:

  1. The <p> tag is used to make a paragraph in semantic HTML. In the snippet above it wraps the entire stament, it helps us see in the code what the element is about, instead of using a general div.

    The <p> tag needs to closed by using <∕p> in order for a paragraph to appear on the page.

  2. The <a> tag is used to insert the link to Google. It only inserts the link to the word "Google," which is why the text is formatted differently. The <a> tag would be considered the child element of the <p> tag since it is below the <a> tag in the hierchy.

    The <a> tag needs to closed by using <∕a> in order for a link to appear on the page.

Attributes of the <a> tag:

  1. "class"

    value = "banana"

    The class attribute allows you to call the element in CSS and style it. It can be assigned to multiple elements in HTML.

  2. "id"

    value = "the-best-link"

    The id attribute works like a label, you can assign it to just one element, targeting that specific element.
